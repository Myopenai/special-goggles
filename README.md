# special-goggles static site

Dieses ZIP enthält eine lauffähige statische Website für GitHub Pages.

## Verwendung

1. Neues GitHub-Repository erstellen (oder vorhandenes nutzen), z.B. `special-goggles`.
2. ZIP-Inhalt **entpacken** und alle Dateien ins Repository hochladen (Root-Verzeichnis).
3. In den Repository-Einstellungen unter **Settings → Pages**:
   - Branch: `main` (oder der Branch, den du verwendest)
   - Folder: `/ (root)`
4. Speichern – GitHub Pages veröffentlicht dann die Seite z.B. unter:
   `https://<dein-username>.github.io/special-goggles/`

Die Startseite ist `index.html`, weitere Inhalte in `TGPA_multilang.html`.
